## Report a Bug

When reporting bugs, please include:

1. **Tool and command** — Which InfraIQ tool were you using?
2. **Expected behavior** — What did you expect to happen?
3. **Actual behavior** — What actually happened?
4. **Steps to reproduce** — How can we recreate the issue?
5. **Environment** — OS, InfraIQ version (`infraiq --version`), cloud provider

Example:

```
Tool: MigrateIQ
Command: migrate scan heroku --app-name myapp
Expected: Successful scan of Heroku app
Actual: Error "Invalid API key" despite valid credentials
Steps: 1. Set HEROKU_API_KEY, 2. Run command
Environment: macOS 14.2, InfraIQ v0.4.3, Heroku
```
