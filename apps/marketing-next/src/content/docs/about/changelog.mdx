# Changelog

All notable changes to InfraIQ will be documented here.

## [0.4.3] - 2026-01-10

### Added
- Dashboard sync integration (`--sync` flag)
- License server migration to Cloud Run + Cloud SQL
- Telemetry infrastructure with Wraith

### Changed
- Improved CLI progress output
- Enhanced error messages

### Fixed
- Various bug fixes and stability improvements

## [0.3.0] - 2025-12-31

### Added
- **Tessera** — AI-powered monolith decomposition
- Beta expiration system (Q1 2026)
- Unified `migrate-and-transform` command
- Docker distribution method

### Changed
- Updated documentation structure
- Improved installation process

## [0.2.0] - 2025-09-04

### Added
- **SecureIQ** — Secret discovery and management
- **DataIQ** — Zero-downtime database migrations
- Heroku to AWS migration improvements
- Self-healing migration engine

### Changed
- Enhanced compliance frameworks
- Improved scan performance

## [0.1.0] - 2025-08-25

### Added
- **VerifyIQ** — Infrastructure verification and security scanning
- **CodifyIQ** — Infrastructure as Code generation
- **MigrateIQ** — Cloud-to-cloud migration automation
- **ComplyIQ** — Compliance automation for SOC2, HIPAA, ISO27001
- Initial CLI framework
- AWS and Heroku provider support
