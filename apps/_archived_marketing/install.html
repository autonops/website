Install InfraIQ

Beta Access: Thanks for joining the InfraIQ beta! This page contains installation instructions. If you have questions, reach out to [email protected].

Quick Install

Run this command in your terminal:

curl -sSL https://install.autonops.io | bash

Prerequisites

  • Python 3.9+ — Check with python3 --version
  • pip — Usually included with Python
  • Cloud credentials — AWS, GCP, Heroku, or Azure CLI configured
  • Terraform 1.0+ — For applying generated configurations

Verify Installation

After installation, verify everything is working:

# Check installation infraiq --version # Run diagnostics infraiq doctor # See all available tools infraiq info

Quick Start Examples

Migrate from Heroku to AWS

# Scan your Heroku app infraiq migrate scan heroku --app-name my-app # Map to AWS services infraiq migrate map scan.json aws # Generate Terraform infraiq migrate generate migration-plan.json --output terraform/

Verify Infrastructure Security

# Quick security scan infraiq verify scan --provider aws --output report.json # Analyze results infraiq verify analyze report.json

Codify Existing Infrastructure

# Discover AWS resources infraiq codify scan aws --region us-east-1 -o scan.json # Generate Terraform infraiq codify generate scan.json --output ./terraform

All Available Tools

  • MigrateIQ — Cloud-to-cloud migration automation
  • VerifyIQ — Security scanning and drift detection
  • CodifyIQ — Convert existing infrastructure to IaC
  • ComplyIQ — SOC2, HIPAA, ISO27001 compliance
  • DataIQ — Zero-downtime database migrations
  • SecureIQ — Secret discovery and management
  • Tessera — AI-powered monolith decomposition

Telemetry

InfraIQ collects anonymous usage data to improve the product. No credentials or personal information are collected. To opt out:

export INFRAIQ_TELEMETRY=false
