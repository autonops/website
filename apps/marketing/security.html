What We Don't Have Access To

  • Cloud credentials â€” Your AWS keys, GCP service accounts, and Heroku tokens stay on your machine
  • Infrastructure data â€” Scan results, resource configurations, and migration plans are stored locally
  • Database contents â€” DataIQ and Tessera analyze schema metadata only, never query actual data
  • Secrets â€” SecureIQ discovers secret locations by name patterns, never reads values
  • Generated code â€” Terraform, Kubernetes manifests, and service scaffolds stay on your machine

Telemetry

InfraIQ collects anonymous usage telemetry to improve the product. This is completely optional and can be disabled.

What We Collect

  • Commands run (e.g., "migrate scan heroku")
  • Execution time and success/failure status
  • Tool versions and OS type
  • Anonymous session ID (random, not tied to identity)

What We Don't Collect

  • IP addresses (stripped at ingestion)
  • Cloud credentials or tokens
  • Resource names, IDs, or configurations
  • Database schemas or data
  • Environment variables or secrets
  • Any personally identifiable information

Opt Out

To disable telemetry:

export INFRAIQ_TELEMETRY=false
